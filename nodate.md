# No date

Saves a fork for the call to the external program `date`.

```bash
# Now
printf "%(%Y-%m-%d %H:%M:%S)T\n" -1

# Unixtime
printf "%(%Y-%m-%d %H:%M:%S)T\n" 1608868800
```
